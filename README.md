# 图片压缩工具

一个简单易用的图片压缩工具，支持将图片压缩至指定大小。

## 功能特点

- 🖼️ 支持多种图片格式（JPG、PNG、BMP、GIF、TIFF）
- 📏 可设置目标文件大小（10KB - 1000KB）
- ⚡ 智能压缩算法，保持最佳图片质量
- 🎯 直观的图形用户界面
- 🔄 实时进度显示

## 环境要求

- Python 3.6+
- 操作系统：Windows、macOS、Linux

## 安装依赖

在运行程序前，请先安装所需的Python包：

```bash
pip install -r requirements.txt
```

## 使用方法

1. 运行 `main.py` 启动程序
2. 点击"浏览"选择要压缩的图片文件
3. 设置目标文件大小（KB）
4. 设置图片质量（百分比）
5. 选择输出文件路径
6. 点击"开始压缩"按钮
7. 等待压缩完成

## 程序界面说明

- **选择图片文件**：选择要压缩的源图片
- **目标大小**：滑动条设置期望的文件大小（10-1000KB）
- **图片质量**：滑动条设置压缩质量（10-100%）
- **输出文件**：设置压缩后图片的保存路径
- **进度条**：显示压缩进度
- **状态标签**：显示当前操作状态

## 技术说明

程序使用PIL（Pillow）库进行图片处理，采用渐进式压缩算法：

1. 首先使用用户设置的质量参数进行压缩
2. 如果文件大小仍超过目标值，自动降低质量参数
3. 重复压缩直到达到目标大小或最低质量限制

## 注意事项

- 压缩过程会保持图片的原始宽高比
- 对于透明图片（PNG），会自动转换为RGB模式
- 建议目标大小不要设置得过小，以免影响图片质量
- 程序会自动优化图片存储格式
